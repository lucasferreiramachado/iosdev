## iosdev
An automation task runner for iOS Development infrastructure

# Getting Started

## Install

```sh
git clone https://github.com/lucasferreiramachado/iosdev
cd iosdev
swift run iosdev install
iosdev --help 
```

## Uninstall

```sh
iosdev uninstall
```
